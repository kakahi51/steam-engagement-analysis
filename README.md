# Analyzing Engagement Patterns on Steam

This project analyzes user engagement patterns on the Steam platform using playtime and popularity metrics.
The analysis focuses on comparing free-to-play and paid games to understand how monetization models impact engagement.

## Dataset
Steam games dataset

## Tools
- Python
- Pandas
- Seaborn
- Matplotlib

## Key Insights
- Free-to-play games show higher engagement potential compared to paid games
- Paid games demonstrate more consistent engagement distribution
- Engagement on Steam is highly skewed toward a small subset of titles
